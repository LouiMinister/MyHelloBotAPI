# MyHelloBotAPI


## CLI 명령어



### 설치

```sh
 $ git clone https://github.com/LouiMinister/MyHelloBotAPI.git
```


### 서버 실행

```sh
 $ npm run start 
```


### 테스트 코드 실행

```sh
 $ npm run test
```
